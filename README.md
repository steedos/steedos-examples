# Steedos Examples

## Getting Started

### Start mongodb & redis service

```bash
cd with-docker-compose
docker-compose up
```

### Start steedos service

```bash
cd ${example-name}
yarn
yarn start
```

## Create as a template

```bash
npx create-steedos-app -e https://github.com/steedos/steedos-examples/tree/main/${example-name}
```
